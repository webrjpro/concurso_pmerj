## Processo de Contribuição

1. Todo código novo deve manter as restrições de segurança do projeto (`lib/security.ts`).
2. NUNCA adicione segredos, senhas ou tokens ao código-fonte.
3. Utilize as rotas de API existentes ou siga o padrão de validação em novas rotas.
4. Rode `npm run build` e `npm run typecheck` antes de submeter PRs.
